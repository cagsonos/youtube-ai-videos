## Descripción

• Explica qué es una API como puente para conectar sistemas y obtener datos externos, esencial para AI agents sin necesidad de código.
• Detalla los componentes de una solicitud HTTP (métodos GET/POST, endpoints, parámetros en query/header/body) y cómo usar cURL con clave API.
• Muestra integración nativa vs. HTTP Request en n8n, un builder no-code visual para configurar llamadas API paso a paso.
• Enseña a usar APIs en workflows de agentes AI con variables dinámicas, credenciales seguras y manejo de respuestas (200 éxito, 400/401/500 errores).
