## Descripción

• Construye un workflow avanzado en n8n para scraping diario de reviews de Trustpilot, usando HTTP para HTML, extracción con CSS e IA para datos y análisis de sentimiento.
• Integra validación de duplicados en Google Sheets y alertas automáticas para reseñas negativas, facilitando el monitoreo continuo.
• Aprende a productizar el flujo con cambios mínimos (prompts e IDs) para venderlo a PYMES por 2.700 euros, mejorando servicios basados en análisis.
• Descubre técnicas para automatizar y escalar scrapers rentables con n8n, ideal para consultores.
