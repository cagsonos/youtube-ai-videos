## Descripción

• Construye un agente de voz AI con n8n y Vapi que llama automáticamente a nuevos leads para calificarlos y recopilar datos clave como motivación, urgencia y presupuesto.
• Flujo completo: formulario web → normalización de teléfono → llamada API → polling de resultados → registro en Google Sheets.
• Configuración detallada de Vapi: prompts dinámicos, structured outputs, manejo de voicemail y números incorrectos.
• Demo en vivo, wireframe de alto nivel y recursos gratuitos para replicar el sistema y optimizar conversiones.
