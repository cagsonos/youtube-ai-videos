## Descripción

- Se explica un agente RAG que conecta 11 Labs, webhook Naden y una base de vectores Pinecone con datos de proyectos.  
- El flujo recibe preguntas por voz, envía la consulta a Naden, recupera respuestas del vector store y las devuelve a 11 Labs.  
- Reutiliza agentes existentes cambiando solo la interfaz.
