## Descripción

• Aprende a construir un agente RAG production-ready en minutos sin código, usando n8n, Pinecone Assistant y OpenRouter para bases de conocimiento 24/7.
• Configura el chat trigger en n8n, conecta el LLM de OpenRouter, añade memoria al agente y prueba respuestas iniciales.
• Sube documentos como PDFs a Pinecone, integra con n8n para queries dinámicas, instrucciones del sistema y citas destacadas.
• Crea una interfaz de chat limpia con Lovable, activa modo producción y prueba el sistema completo con documentos reales.
