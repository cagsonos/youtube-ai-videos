## Descripción

**Resumen:**
*   El video demuestra cómo crear un sistema de IA de voz para HVAC usando Claude Code, Retail AI y Naden, sin necesidad de codificación.
*   Claude Code genera un agente de voz funcional, incluyendo flujos de trabajo de programación, verificación de disponibilidad, reserva de citas y análisis post-llamada.
*   El creador explica el proceso paso a paso, mostrando la integración con Google Calendar y configuraciones para notificaciones por email y Slack.
