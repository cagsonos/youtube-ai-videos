## Descripción

• n8n introduce tablas de datos nativas para almacenar y gestionar información internamente, sin bases de datos externas ni llamadas API constantes.
• Aprende a configurar tablas, poblarlas con datos (incluso importando de hojas externas), filtrar filas y conectarlas a workflows o agentes AI para análisis, como calcular ingresos de ventas.
• Descubre cómo usar AI para respuestas personalizadas basadas en tablas, con demos de filtrado y análisis de datos de ventas.
• Comparación de velocidad: tablas nativas superan a Google Sheets en rendimiento, evitando latencia y límites de API; pros, contras y cuándo optar por bases de datos tradicionales.
