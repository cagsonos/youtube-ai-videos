## Descripción

• Implementa un buffer con Redis en n8n para concatenar mensajes de WhatsApp enviados en ráfagas, evitando respuestas múltiples del bot.
• Configura una ventana de 7 segundos con un nodo Switch para validar usuario y agrupar mensajes breves.
• Usa expresiones JSON para validaciones inteligentes, concatenación final y limpieza automática de datos temporales en RAM.
• Importa plantilla JSON lista, integra con Evolution API y prueba en vivo el flujo completo.
