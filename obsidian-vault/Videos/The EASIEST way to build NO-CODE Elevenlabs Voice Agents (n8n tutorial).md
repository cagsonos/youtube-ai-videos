## Descripción

• Aprende a crear agentes de voz AI con ElevenLabs para llamadas entrantes y salientes sin código, usando solo 3 workflows de n8n.
• Configura integración con Twilio para números de teléfono, pruebas de inbound y outbound con una sola petición HTTP.
• Construye workflows post-llamada para analizar transcripciones, registrar en Google Sheets y activar acciones como SMS o email.
• Implementa variables dinámicas, transferencia a humanos, RAG y edición de prompts del sistema para personalización avanzada.
