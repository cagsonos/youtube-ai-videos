## Descripción

• Aprende a habilitar **Extended Thinking** en n8n para modelos Anthropic como Claude Sonnet 4.5, obteniendo tokens de reflexión separados del texto final.
• Compara resultados con OpenRouter, Think Node y nodos OpenAI, que no devuelven tokens de reflexión.
• Usa **Chain-of-Thought** con etiquetas XML para emular reflexión, ideal si el presupuesto es menor a 1.024 tokens.
• Extrae **Reasoning Effort** de ChatGPT para mejorar el razonamiento en flujos automatizados.
