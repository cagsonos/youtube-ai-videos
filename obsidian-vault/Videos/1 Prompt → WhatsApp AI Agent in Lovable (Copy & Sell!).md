## Descripción

• Construye un agente IA completo para WhatsApp que gestiona citas automáticamente con un solo prompt en Lovable, sin código ni herramientas como n8n.
• Integra Twilio para WhatsApp, configura webhooks y crea un dashboard con mensajes, citas y ajustes usando Supabase.
• Aprende a probar el agente en tiempo real, ajustar respuestas con prompts y empaquetar el sistema para vender a negocios locales por $1K-$3K.
• Explora recursos como plantillas de prompts y curso BuildLoop para desarrollar apps IA production-ready.
