## Descripción

• Aprende a pasar de Sandbox a modo Live en WhatsApp API de Meta sin verificación empresarial (nivel básico: 250 conversaciones/24h).
• Crea políticas de privacidad gratuitas con ChatGPT y Google Docs para cumplir requisitos.
• Libera tu número real de WhatsApp Business, agrégalo a la API y verifícalo por SMS sin perderlo.
• Configura método de pago, entiende costos (gratis para iniciados, pago por salientes) y actualiza credenciales en N8N.
