## Descripción

• Aprende qué es Claude Code, una extensión de VS Code para crear workflows de N8N usando prompts en lenguaje natural.
• Configura el setup completo: instala la extensión, crea un archivo CLAUDE.md y configura MCPs/skills de N8N.
• Integra la API de N8N y genera workflows funcionales con un solo prompt, aplicando mejores prácticas.
• Ve un ejemplo real de workflow final, como un agente de WhatsApp automatizado.
