## Descripción

• **Workflow con template Claude Code:** Usa un boilerplate preconfigurado para manejar estructura, autenticación y dependencias en apps Next.js con Supabase y ShadCN, evitando reconstruir bases técnicas.
• **Configuración inicial:** Instala Node.js 20+, crea cuenta Supabase, clona repo de GitHub, configura .env.local y genera usuario de prueba.
• **Desarrollo de features:** Agrega funcionalidades como upload de facturas, parseo con API de Claude y almacenamiento en Supabase, enfocando IA solo en nuevas características.
• **Beneficios clave:** Construye 10x más rápido, reduce errores, mayor control y accesible para no-desarrolladores en proyectos AI.
