## Descripción

- Los webhooks son mensajes HTTP con payload estructurado.  
- N8n captura webhooks con **Catch Hook** y los envía con **HTTP Request**.  
- Se utilizan para integrar apps, por ej. email → CRM.  
- Respuesta: JSON, texto, binario o HTML, generando micro‑APIs.  
- Ejemplos: API de historia por nombre, redirección con datos y frontend estático.
