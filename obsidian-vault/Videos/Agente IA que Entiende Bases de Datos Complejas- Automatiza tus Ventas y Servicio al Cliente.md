## Descripción

• Sistema de 3 agentes IA en n8n: Orquestador dirige consultas, FAQ responde con base vectorial (Supabase), y SQL maneja bases complejas (PostgreSQL) para automatizar ventas y servicio.
• Funcionalidades clave: registrar clientes, crear/actualizar pedidos, validar direcciones y responder FAQs en tiempo real, integrado con WhatsApp via ChatWoot.
• Tutorial paso a paso: lógica del workflow, vectorización de PDFs/FAQs, conexión a DB, pruebas y visualización con NocoDB, ideal para restaurantes y e-commerce.
