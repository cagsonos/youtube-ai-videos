## Descripción

• Gemini File Search es un servicio RAG gestionado que procesa archivos (con OCR y embeddings), pero requiere pipelines externos para manejar duplicados, versionado y metadatos avanzados.
• Revela 5 limitaciones clave: necesidad de data pipelines, chunking básico sin Markdown, desafíos en metadatos, caja negra de rango medio y vendor lock-in con implicaciones de privacidad.
• Compara precios con OpenAI, muestra 3 integraciones con n8n (pros/contras) y workflows completos de ingesta e inferencia para entornos de producción.
• Veredicto: útil pero no un "game-changer" que reemplace RAG personalizado; ideal para casos simples con precauciones en personalización y dependencia de Google.
