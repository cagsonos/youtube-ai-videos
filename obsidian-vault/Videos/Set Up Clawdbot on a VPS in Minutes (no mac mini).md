## Descripción

• **Ventajas del VPS:** Explica por qué usar un VPS como Hostinger es ideal para Clawdbot, evitando la necesidad de un Mac Mini.
• **Configuración inicial:** Guía paso a paso para crear el servidor VPS, instalar Node.js, Clawdbot y configurar usuario dedicado.
• **Onboarding y conexiones:** Configura claves API (Anthropic y Telegram), gateway, tunnel y conexión para operación 24/7.
• **Seguridad esencial:** Advierte sobre riesgos como inyección de prompts y acceso a datos sensibles, recomendando entornos seguros y control de permisos.
