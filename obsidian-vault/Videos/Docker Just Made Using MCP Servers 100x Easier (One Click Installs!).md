## Descripción

• Docker presenta el MCP Catalog, un lugar único para encontrar e instalar servidores MCP pre-empaquetados en contenedores seguros con un solo clic, simplificando su uso con agents de IA.
• Aprende a usar el Docker MCP Toolkit para agregar servidores (YouTube, Slack, GitHub, Obsidian) directamente a clientes como Claude Desktop, Codex o Claude Code.
• Explora flujos agenticos completos: transcripción de videos, resúmenes, creación automática de issues y PR en GitHub, orquestados por el MCP Gateway sin carga continua.
• Conecta servidores MCP a agents personalizados mediante el MCP Gateway, con ejemplos en n8n y LiveKit para workflows avanzados.
