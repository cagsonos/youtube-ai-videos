## Descripción

- Se automatiza la extracción de datos de PDFs, currículos, facturas y etiquetas de envío mediante un flujo en “Nad” (orquestador equivalente a Zapier).  
- Se detecta el tipo de documento con OpenAI Vision o extracción de texto nativa.  
- OpenAI devuelve resultados estructurados (“JSON schema”), facilitando su integración programática.  
- Los datos se guardan automáticamente en Google Sheets o sistemas elegidos.  
- El proceso es rentable: < 2 cents/documento, reduce trabajo manual y errores.
