## Descripción

• Construye un agente RAG multimodal en n8n para indexar PDFs con texto, imágenes y tablas usando OCR Mistral en formato markdown LLM-friendly.
• Extrae y anota contenido de documentos, incluyendo análisis de imágenes con modelos de visión, para un contexto profundo.
• Vectoriza datos con embeddings OpenAI, almacénalos en Supabase y sube imágenes con sus anotaciones al storage.
• Desarrolla un chatbot que consulta la base vectorial, usa GPT-4 para generar respuestas y renderiza imágenes directamente en la salida.
