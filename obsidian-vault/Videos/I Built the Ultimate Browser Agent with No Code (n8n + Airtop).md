## Descripción

• Construye un agente AI de navegador sin código en n8n con Airtop, que inicia browsers remotos, navega, escribe y completa tareas como un humano.
• Configura integración con OpenRouter/Claude 3.5, system prompt y herramientas clave: "Start Browser", "Type", "Query" y "Think" para análisis visual.
• Aprende ejemplos prácticos: búsqueda en Google, extracción de datos y comparación de precios en BestBuy, con enlace en vivo para monitoreo.
• Accede a plantilla gratuita, autenticación Airtop y detalles de precios con descuento (código NateHalfOff).
