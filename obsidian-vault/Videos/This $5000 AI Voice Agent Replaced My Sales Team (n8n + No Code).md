## Descripción

• Construye un agente de voz AI production-level para ventas outbound que califica leads, analiza llamadas y automatiza follow-ups, reemplazando un equipo de ventas.
• Configura Twilio para telefonía, ElevenLabs para voces personalizadas y n8n como backend sin código para manejar llamadas y webhooks.
• Integra Airtable como CRM para triggers de leads, fetch de datos de clientes, análisis post-llamada y acciones automáticas como agendar reuniones.
• Crea dashboards atractivos para monitoreo y visualiza el flujo completo desde la demo inicial hasta el análisis final.
