## Descripción

• Configura un disparador de errores en N8N para supervisar y capturar fallos en flujos de trabajo automáticamente.
• Usa un nodo de código para convertir los datos de error en formato JSON legible y estructurado.
• Integra notificaciones con Telegram mediante bot token y chat ID para alertas en tiempo real.
• Obtén el ID de chat con un trigger "on message" y guárdalo para reutilizar en el flujo de notificaciones.
