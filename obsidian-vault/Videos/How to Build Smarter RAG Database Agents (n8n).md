## Descripción

• Aprende a construir un agente IA en n8n que consulta bases SQL con lenguaje natural, extrayendo esquemas y claves foráneas dinámicamente para generar consultas complejas.
• Descubre cómo optimizar el system prompt y herramientas para mejorar precisión en joins, filtros y razonamiento con modelos como GPT-4-mini.
• Explora técnicas de seguridad contra inyección SQL usando usuarios de solo lectura, validación y enfoques como vistas desnormalizadas o consultas predefinidas.
