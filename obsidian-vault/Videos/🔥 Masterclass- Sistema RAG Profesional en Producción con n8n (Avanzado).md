## Descripción

• Implementa un agente RAG profesional en n8n para producción usando Postgres vectorial con Supabase, cubriendo conceptos como embeddings, chunks y bases de datos vectoriales.
• Aprende técnicas avanzadas de inserción de documentos: generación de claves hash para evitar duplicados, adición de metadatos y actualización de contenidos.
• Domina consultas en Supabase vía API o servidor propio, incluyendo reclasificación (reranking) con Cohere para resultados óptimos.
• Demostración práctica completa: desde carga de documentos hasta agente IA RAG en entornos reales.
