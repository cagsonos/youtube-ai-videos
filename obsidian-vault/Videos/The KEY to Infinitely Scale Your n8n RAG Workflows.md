## Descripción

• El video aborda desafíos de escalado en workflows RAG de n8n, como sobrecargas de memoria y límites de API, resueltos con un orquestador que procesa lotes paralelos de 50 archivos vía webhooks.
• Aprenderás a construir el orquestador con Supabase para rastrear ejecuciones padre/hijo, manejo de errores con reintentos automáticos y SFTP para descargas seguras.
• Se demuestra la adaptación de workflows de ingesta, preferencia de webhooks sobre subworkflows para mejor rendimiento, y optimizaciones como desactivar logs de éxitos.
• Claves para escalar a 5.000 archivos/hora: capas de reintentos, configuración de Supabase y lecciones prácticas para alta fiabilidad.
