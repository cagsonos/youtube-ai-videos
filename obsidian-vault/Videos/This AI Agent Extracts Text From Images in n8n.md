## Descripción

- Telegram envía una imagen de factura a la API.  
- Una OCR convierte la imagen en texto y extrae campos claves.  
- Los datos se guardan en una base de datos relacional.  
- La imagen original se sube a Google Drive para referencia.  
- Un agente de IA responde con resumen, enlaces y avisos.
