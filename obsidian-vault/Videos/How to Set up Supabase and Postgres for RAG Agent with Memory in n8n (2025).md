## Descripción

- Se conecta Postgres como memoria de chat en Naden.  
- Se usa Supabase como vector store con embeddings.  
- Se configuran credenciales: host, usuario, contraseña, puerto y secret para ambos servicios.  
- Se insertan documentos en Supabase, procesados por un splitter y embeddings.  
- Se recupera información mediante RAG, actualizando el historial en Postgres.
