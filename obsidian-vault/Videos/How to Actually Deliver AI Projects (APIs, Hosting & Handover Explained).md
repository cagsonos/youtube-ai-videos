## Descripción

• **Hosting de workflows:** Aprende a recomendar que el cliente hostee n8n de forma self-hosted para cumplir con compliance y evitar problemas de licencias en SaaS.
• **Seguridad y protección de datos:** Descubre cómo encriptar credenciales, autenticar webhooks y usar self-hosting para garantizar soberanía de datos.
• **Gestión de API keys:** Entiende cómo transferir el control de las claves API al cliente, minimizando riesgos y facilitando la independencia.
• **Handover del workflow:** Conoce el proceso de entrega: QA con datos reales, documentación en Loom, backups JSON y soporte post-lanzamiento para clientes felices.
