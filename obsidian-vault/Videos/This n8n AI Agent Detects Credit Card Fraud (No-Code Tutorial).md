## Descripción

• Configura un trigger de Gmail en n8n para analizar correos de transacciones bancarias y extraer datos clave como monto, fecha, comerciante y número de tarjeta.
• Implementa un agente IA que utiliza Pinecone como base de datos vectorial para comparar transacciones con el historial y detectar anomalías de fraude.
• Aprende a configurar herramientas y prompts del agente IA para evaluar riesgos y generar salidas estructuradas.
• Procesa los resultados en Google Sheets, almacena análisis y envía alertas automáticas según el nivel de riesgo.
