## Descripción

- El video contrasta AI agents (flexibles pero ineficientes para procesos lineales) con workflows determinísticos, que priorizan fiabilidad, bajo costo y escalabilidad en herramientas de IA.
- Ventajas de workflows: ejecución secuencial reduce errores y llamadas API innecesarias.
- Aplicaciones prácticas: soporte al cliente vía RAG en Pinecone y análisis de gráficos bursátiles en n8n.
