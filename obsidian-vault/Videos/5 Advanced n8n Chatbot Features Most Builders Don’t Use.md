## Descripción

• Aprende a rastrear sesiones ID para gestionar memoria, estado (como bans por jailbreaks) y rutas conversacionales en n8n.
• Implementa guardrails que detectan intentos de jailbreak y banean usuarios vía tablas de sesiones.
• Usa RAG nativo de OpenAI con búsqueda web restringida y vector stores de archivos para respuestas contextuales como FAQs.
• Configura nodos de chat y subworkflows para streaming de respuestas, mensajes intermedios y herramientas reutilizables en sistemas multi-agente.
