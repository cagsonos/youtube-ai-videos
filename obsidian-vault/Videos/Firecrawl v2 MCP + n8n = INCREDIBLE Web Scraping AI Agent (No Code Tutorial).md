## Descripción

• Aprende a usar Firecrawl v2 para convertir sitios web en Markdown compatible con IA, con funciones como mapear, raspar, rastrear y extraer datos.
• Configura el servidor MCP de Firecrawl con n8n para crear un agente de web scraping sin código, simplificando a acciones de listar y ejecutar herramientas.
• Demostración práctica: automatiza la extracción de precios y reseñas de páginas de Amazon para análisis competitivo y generación de leads.
• Descubre beneficios como manejo de sitios JavaScript-heavy, evasión de bloqueos y datos estructurados para IA, ideal para agencias y automatizaciones.
