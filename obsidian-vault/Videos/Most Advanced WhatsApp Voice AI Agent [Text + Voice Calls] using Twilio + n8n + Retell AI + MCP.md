## Descripción

• Configura un número WhatsApp en Twilio y construye un agente AI de texto con n8n, integrando GPT para manejar consultas.
• Almacena conversaciones en Supabase (Postgres), crea subflujos para reservas en Google Calendar, base de conocimiento y envíos de emails.
• Desarrolla el agente de voz AI en Retell AI, usando MCP para exponer herramientas y SIP/TwiML para conectar llamadas vía n8n.
• Implementa webhooks, flujos completos de llamadas (Twilio → n8n → Retell AI) y realiza pruebas end-to-end para un agente de producción listo.
