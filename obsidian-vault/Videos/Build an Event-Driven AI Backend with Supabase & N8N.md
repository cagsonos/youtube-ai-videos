## Descripción

• Construye un backend impulsado por eventos con webhooks de Supabase que activan flujos en n8n para automatizaciones AI escalables.
• Aprende a disparar workflows RAG al subir o eliminar archivos en Supabase Storage, ingiriendo contenido y actualizando registros.
• Crea un "Recipe Generator" que genera texto estructurado, etiquetas e imágenes al insertar filas en la base de datos, guardándolos en Supabase.
• Conecta OpenAI y Supabase sin código backend, usando n8n o ActivePieces para lógica de eventos.
• Explora la arquitectura event-driven con Postgres como vector database para low-code AI automation.
