## Descripción

- Se instala Nano localmente mediante **npx** (se requiere Node.js).  
- Se añade el paquete **MCP nodes** para integrar servidores MCP en los flujos de trabajo de Nano.  
- Se crean credenciales con **API key** y variables de entorno para conectar al servidor MCP.  
- El flujo combina un agente LLM con acciones MCP, usando el **tool‑Tavly** para listar y ejecutar tareas.  
- Se demuestra la consulta de precios de criptomonedas (ETH‑USD) pasando resultados de Tavly al LLM.
