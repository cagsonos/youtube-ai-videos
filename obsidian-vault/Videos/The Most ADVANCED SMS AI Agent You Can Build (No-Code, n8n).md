## Descripción

• Aprende a construir un agente AI SMS avanzado sin código usando n8n, Twilio, GPT-4.1 y memoria en Postgres.
• Configura el agrupamiento de mensajes en lotes de 5-10 segundos con Google Sheets para procesamiento eficiente.
• Integra búsqueda en tiempo real vía API de Perplexity (filtrando dominios) y genera respuestas ≤160 caracteres para soporte/ventas.
• Conecta Twilio para envíos automáticos de respuestas y explora demos en vivo de inbound/outbound SMS.
