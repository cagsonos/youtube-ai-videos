## Descripción

• Explica cómo construir un agente AI en n8n para responder preguntas en lenguaje natural sobre bases de datos SQL, ya que el agente predeterminado fue eliminado.
• Guía paso a paso para configurar una base de datos Postgres gratuita en Supabase, crear tablas y cargar datos de métricas de videos y comentarios.
• Detalla la creación del workflow completo en n8n que integra IA para generar y ejecutar consultas SQL automáticamente.
• Demuestra pruebas exhaustivas: filtros básicos y agregaciones (conteo, promedio), joins multi-tabla y análisis temporal para validar insights reales.
