## Descripción

- Integromat controla un servidor mCP para ejecutar herramientas IA, como Brave Search.  
- Un formulario Tally envía consultas por webhook a Integromat.  
- La búsqueda Brave devuelve JSON; OpenAI GPT‑3.5 crea instrucciones resumidas.  
- Resultados se guardan en una base Notion y se notifica por Gmail.
