## Descripción

• Aprende a habilitar y configurar webhooks en Supabase para disparar solicitudes HTTP o funciones Edge ante operaciones CRUD en tablas.
• Configura webhooks seleccionando tabla, evento (insert, update) y URL o función Edge para automatizar procesos.
• Prueba inserciones (muestra datos nuevos) y actualizaciones (incluye registro viejo y nuevo) directamente en la UI.
• Integra funciones Edge con webhooks para lógica personalizada, como enviar usuarios a un CRM.
• Recopilación: webhooks facilitan automatizaciones sin complicaciones en Supabase.
