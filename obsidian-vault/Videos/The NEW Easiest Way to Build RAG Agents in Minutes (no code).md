## Descripción

• Demostración de la forma más fácil y rápida para construir agentes RAG sin código, usando Pinecone Assistant y n8n, sin preprocesamiento manual.
• Configuración paso a paso: carga automática de documentos en Pinecone (ingesta, splitting e indexado) y conexión a n8n para respuestas precisas.
• Uso de la API de Pinecone con "include highlights" para obtener citas de página y fragmentos exactos de PDFs.
• Mejoras al agente RAG, comparación de rendimiento y costos vs. métodos vectoriales tradicionales, ideal para principiantes y prototipado rápido.
