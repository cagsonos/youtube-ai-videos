## Descripción

• Construye un agente AI multi-usuario seguro en n8n para e-commerce, con aislamiento estricto de datos entre clientes y acceso a historial de órdenes, facturas y tickets.
• Aprende 7 estrategias clave de seguridad: proxy WordPress para webhooks, JWT con expiración corta, Row-Level Security (RLS) en Supabase y MFA para operaciones sensibles.
• Configura verificación de origen, privilegios mínimos, monitoreo, anonimización de datos y cumplimiento con GDPR/CCPA.
• Cubre problemas de n8n Embed, multi-user vs multi-instance, y requisitos de licencias n8n para uso comercial.
