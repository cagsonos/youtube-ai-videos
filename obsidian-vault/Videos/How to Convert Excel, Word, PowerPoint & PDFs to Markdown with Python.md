## Descripción

- **MarkItDown**: Biblioteca Python de Microsoft para convertir documentos Office (Excel, Word, PDF), ZIP y URLs de YouTube a Markdown, ideal para entrenar LLMs o custom GPTs en ChatGPT.
- **Uso simple**: Instalar con pip; método de conversión extrae texto estructurado (ej. tablas en Excel, transcripciones de videos).
- **App web**: Herramienta Streamlit gratuita para subir archivos y descargar Markdown sin código, mejorando accesibilidad para aprendizaje en IA.
