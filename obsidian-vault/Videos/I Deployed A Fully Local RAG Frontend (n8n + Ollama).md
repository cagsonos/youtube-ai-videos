## Descripción

• Demostración de un frontend RAG local similar a NotebookLM, desplegado con Docker usando Ollama, Supabase y n8n, sin dependencias cloud para privacidad y control total.
• Stack clave: ingesta de PDFs con embeddings, LLM Qwen3 8B via Ollama, transcripción Whisper local y TTS Coqui para podcasts automáticos.
• Orquestación visual sin código en n8n, con workflows optimizados para hardware limitado (ej. 8GB VRAM) y manejo de citas en Q&A.
• Guía paso a paso de setup (desde 10:46), verificación y repositorios Git para personalización y réplicas.
