## Descripción

• El video demuestra cómo usar n8n-mcp con Claude Code para crear workflows en n8n directamente desde prompts, sin código JSON.
• Tutorial paso a paso para configurar el servidor n8n-mcp y probarlo en tareas simples (chatbot), medianas (newsletter con websearch) y complejas (búsqueda de empleos en LinkedIn).
• Aprenderás las ventajas como acceso a docs y plantillas, limitaciones en tareas avanzadas que requieren ajustes manuales, y evaluación final de su efectividad.
• Incluye recursos para empezar con n8n, comunidad Skool y GitHub del proyecto.
