## Descripción

• Aprende a automatizar recordatorios de WhatsApp para citas usando n8n, Baserow (base de datos gratuita) y Cal.com, con bucles que calculan tiempos y evitan duplicados.
• Configura el método oficial con ManyChat para plantillas aprobadas (texto, audio, video), integrándolo vía webhook con n8n.
• Explora el método no oficial con Evolution API, generando mensajes personalizados con ChatGPT (texto), ElevenLabs (audio) y HeyGen (video clonado).
• Obtén códigos, plantillas gratuitas y timestamps para replicar el sistema completo sin código avanzado.
