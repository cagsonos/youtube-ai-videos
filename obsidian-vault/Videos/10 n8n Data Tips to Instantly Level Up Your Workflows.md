## Descripción

• Aprenderás a usar expresiones como `now()`, `executionMetadata` y `workflowName` para simplificar el logging y formateo de datos en workflows de n8n.
• Descubrirás el nodo *Edit Fields* para centralizar cambios de variables y mantener consistencia al escalar flujos.
• Explorarás nodos *Code* después de APIs para analizar JSON anidado y procesar datos de inmediato.
• Dominarás *ReturnAll* y loops para reconstruir objetos y manejar paginación, como en tickets de HubSpot.
• Utilizarás nodos *Do Nothing* y *Merge* para organizar flujos y combinar caminos paralelos de forma limpia.
