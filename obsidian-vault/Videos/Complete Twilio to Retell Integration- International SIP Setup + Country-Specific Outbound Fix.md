## Descripción

• Tutorial paso a paso para comprar un número Twilio, configurar paquetes regulatorios y habilitar voz/SMS en países específicos.
• Configuración de tronco SIP elástico: activar transferencias, establecer URI de terminación, crear listas de credenciales y URL de origen.
• Integración con Retell AI: enlazar número con credenciales y URI país-específicos (ej. au1-retell.pstn.twili.com), pruebas de llamadas entrantes/salientes.
• Solución de problemas en llamadas salientes internacionales, como códigos cortos para Nueva Zelanda y métodos alternativos de configuración SIP.
