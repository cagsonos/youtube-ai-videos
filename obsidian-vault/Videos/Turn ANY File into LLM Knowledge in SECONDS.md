## Descripción

• Docling es un framework open-source que extrae texto, tablas e imágenes de cualquier archivo (PDF, Word, audio, video) generando Markdown optimizado para RAG en LLMs.
• Aprenderás a instalar y usar Docling con ejemplos prácticos: parsing de PDFs complejos, manejo de múltiples formatos y extracción de texto de audio.
• Descubre el chunking híbrido, que segmenta contenido por similitud semántica para consultas precisas en bases vectoriales.
• Recursos clave: repositorio GitHub, documentación, agente RAG listo como template y taller en Dynamous.
