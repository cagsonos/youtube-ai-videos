## Descripción

• La API de Responses de OpenAI integra herramientas nativas como búsqueda web y vector stores en agentes IA de n8n, eliminando prompts complejos y pipelines de datos.
• Configuración sencilla en n8n: activa el toggle, ingresa API key y ajusta opciones como contexto, dominios restringidos y filtros, sin código.
• Setup de búsqueda web para datos actualizados (ej. récords deportivos) y file search para consultas en archivos con costos por almacenamiento.
• Opciones adicionales mejoran precisión en respuestas, como reglas de golf, potenciando la capacidad de los agentes IA de forma rápida y eficiente.
