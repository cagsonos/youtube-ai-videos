## Descripción

• **Overview de HuggingChat**: Interfaz gratuita y open-source para chatear con múltiples modelos open-source, similar a ChatGPT, con OmniRouter para enrutamiento inteligente.
• **Exploración de HF MCP server y Dynamic Spaces**: Aprende a usar el servidor MCP de Hugging Face y espacios dinámicos para interacciones avanzadas.
• **Búsqueda y documentación en chat**: Descubre cómo buscar modelos, documentación y usar el nuevo chat bubble para docs directamente desde la conversación.
• **Servidores personalizados y pricing**: Conecta tus propios servidores MCP y entiende el modelo de uso gratis inicial, luego pay-as-you-go asequible.
