## Descripción

• Construye un agente de voz AI completo en Retell AI para un gimnasio MMA, desde scraping web hasta optimización de KB con Claude (XML/markdown).
• Configura agente single prompt con GPT-4, settings globales (chunks=3, similarity=0.6), mensaje de apertura y prompt universal para flujos conversacionales.
• Aprende testing en tiempo real (audio/chat) para verificar lógica, precisión, pronunciación y latencia (<2s), refinando prompts y transiciones.
• Verifica respuestas antes de producción y explora integraciones avanzadas como análisis post-llamada y Twilio.
