## Descripción

• Tutorial paso a paso para crear un agente de voz AI con una sola prompt en Retell AI, ideal para principiantes y prototipos rápidos como confirmaciones de citas o encuestas.
• Cuándo usar single prompt vs. agentes multi-estado, sus limitaciones y mejores casos de uso para minimizar alucinaciones.
• Estructura del prompt con fórmula de 6 componentes, inclusión de variables dinámicas (tiempo, nombre de usuario) y configuraciones de voz, modelo (Gemini 2.0 Flash) y transcripción.
• Demostración en vivo: construcción de un agente de confirmación de citas desde cero, más setup post-llamada (resúmenes, etiquetas, voicemails).
