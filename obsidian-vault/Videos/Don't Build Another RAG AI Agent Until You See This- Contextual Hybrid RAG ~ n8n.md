## Descripción

• • Explica los 3 niveles de RAG: Tradicional (búsqueda básica), Contextual (mejora con contexto de chunks) y Contextual Híbrido (combina vectores densos/dispersos para mayor precisión).
• • Implementa Contextual Hybrid RAG basado en artículo de Anthropic, usando n8n para extraer chunks, generar embeddings y almacenarlos en Pinecone.
• • Detalla el proceso de consulta: crea vectores de la pregunta, realiza búsqueda híbrida, aplica re-ranker para ordenar resultados y generar respuestas óptimas.
• • Muestra configuración de índice en Pinecone, ejecución del workflow, contextualización de chunks y revisión de logs para depuración.
