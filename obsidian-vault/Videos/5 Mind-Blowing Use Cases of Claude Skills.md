## Descripción

• Claude Skills son herramientas personalizadas en IA (archivos Markdown) que automatizan tareas repetitivas, evitan la degradación de outputs y ahorran tokens frente a Projects.
• Diferencias clave: Skills para instrucciones fijas y consistentes; Projects para contextos dinámicos; framework práctico para elegir entre ambos.
• Anatomía de una Skill: estructura con instrucciones claras, recursos opcionales y ejemplos para precisión.
• 5 casos de uso plug-and-play: guías de marca, calculadora de puntuación de leads (con Excel), generador de informes de clientes, toma de decisiones estratégicas y analizador de datos de encuestas.
