## Descripción

• Configuración inicial de VPS en Hostinger con Dokploy para gestionar aplicaciones de forma profesional.
• Instalación de n8n usando Docker Compose, PostgreSQL (evitando SQLite) y ajustes de rendimiento.
• Configuración de dominio propio con HTTPS, credenciales API y habilitación de integraciones externas.
• Creación de agente IA conectado a PostgreSQL, más instalación de herramientas como Qdrant, Redis y Supabase.
